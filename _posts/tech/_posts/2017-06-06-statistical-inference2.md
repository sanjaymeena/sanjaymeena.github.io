## the normal distribution

If X is a normal random variable with mean $$ \mathsf { \mu} $$ and variance $$ \mathsf { \sigma^2} $$  , i.e., <b>X </b> ~  $$ \mathsf {  N(\mu,\sigma^2)  } $$   ,  
then the random variable Z defined as  $$ \mathsf { Z = (\frac {X - \mu) } {\sigma}  } $$ is normally distributed with mean 0 and variance 1,

 Z ~ N(0, 1)  (Z is standard normal.)

The converse is also true. If Z is standard normal, Z ~ N(0,1), then the random variable X defined as  $$ \mathsf { X = \mu + \sigma*Z} $$  is normally distributed with mean $$ \mathsf { \mu} $$  and variance $$ \mathsf { \sigma^2} $$ , i.e.

X ~ N $$ \mathsf { (\mu, \sigma^2)} $$ 


These formulae allow us to easily compute quantiles (and thus percentiles) for ANY normally distributed variable if we know its mean and variance.
